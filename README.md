# Docker image that includes common network debugging tools

## Usage

```bash
docker run --rm -it ghcr.io/simonrw/docker-debug:main
```

[Release page](https://github.com/simonrw/docker-debug/pkgs/container/docker-debug)

## Contents

* `dig`
* `ip`
* `ping`
* `route`
* `curl`
* `python3`
* `python3-boto3`
